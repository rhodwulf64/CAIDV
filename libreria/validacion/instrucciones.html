Pluing de Validación para Jquery, Desarrollado por Leonardo Melendez.

Es muy Simple!
* Copia la Carpeta "js" en el directorio principal de tu pagina.

* Agregar estas lineas de codigo a tu pagina, si ya tienes Jquery no agregues la primera linea.:
<script type="text/javascript" src="js/jquery-1.7.min.js"></script>
<script type="text/javascript" src="js/lemez_validacion1.1.js"></script>
<script type="text/javascript">
$(function(){
$("#campo1").lemez_aceptar("texto");
});
</script>

* Imaginemos que en tu pagina tienes un campo de texto el cual deseas validar:
<input type="text">

*Agregarle un ID a la etiqueta input quedando de la siguiente manera:
<input type="text" id="campo1" />

*Agregale un boton tipo BUTTON y un formulario quedado asi:

<form>
<input type="text" id="campo1" />
<button>Probar</button>
</form>

LISTTOOOOOOO!!! si quieres cambiar el tipo de validación solo reemplaza la palabra texto en la siguiente linea:

$("#campo1").lemez_aceptar("texto");

por cualquiera de estas:

TIPO DE VALIDACIÓN DESCRIPCIÓN
texto Acepta solo Letras Mayusculas o Minusculas.
numero Solo Numeros
correo Valida un correo electronico
todo Acepta todo tipo de caracteres.
texto_especial Solo Acepta Texto comas(,) y puntos (.)
texto_numero Acepta texto y numeros
numero_especial Solo se acepta Numeros con puntos(.) y comas(,)
url formato http://leonardo.com
telefono formato 0255-7778888 ó 255-7778888
ip formato 192.168.1.1
fecha Incluye años Bisiestos y evalua terminaciones en 30 o 31 de cada mes
contraseña Entre 8 a 10 caracteres incluye mayuscula, minuscula y numeros
/EXPRESIÓN REGULAR/ Para validaciones Avanzadas.



Si quieres que sea un campo obligatorio agregale el id del boton asi:
$("#campo1").lemez_aceptar("texto","id_boton");
Si son varios:
$("#campo1").lemez_aceptar("texto","id_boton,id_boton2");
Validar varios campos:
$("#campo1").lemez_aceptar("texto");
$("#campo2").lemez_aceptar("texto");
$("#campo3").lemez_aceptar("texto");
